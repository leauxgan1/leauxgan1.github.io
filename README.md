# Portfolio Site
This is the portfolio site for Logan Langdon created using the static site generator [Zine](https://zine-ssg.io/).
The styling for this site makes use of Open-Props for a consistent design system.
As new projects and blog posts are created, this site will be updated to show them off here.

# Purpose and topics
The blog featured on this site is centered on web development topics such as rendering strategies, SEO, and accessibility features and patterns.
Most commonly, comparisons of existing technologies and explorations of new technologies will be discussed and analyzed.
